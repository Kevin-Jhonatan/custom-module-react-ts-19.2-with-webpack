# React 19 + TypeScript Drupal Custom Module

A Drupal custom module that integrates **React 19**, **TypeScript**, **Webpack**, and **Bun**.

## Compatibility

- Drupal 8, 9, 10 and 11
- Drupal 9
- Drupal 10

## Installation

Clone the repository:

```bash
git clone git@github.com:Kevin-Jhonatan/custom-module-react-ts-19.2-with-webpack.git
````

Go to the module directory:

```bash
cd custom-module-react-ts-19.2-with-webpack
```

Install dependencies:

```bash
bun install
```

Start development mode:

```bash
bun run start
```

## Production Build

```bash
bun run build
```

## Tech Stack

* React 19.2.7
* React DOM 19.2.7
* TypeScript 5
* Webpack 5
* Bun
* Drupal Custom Module

## Notes

* `bun run start` runs Webpack in watch mode.
* `bun run build` generates an optimized production build.
* The module can be used as a starting point for React applications inside Drupal custom modules.
